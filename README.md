# Local File Manager

> 一个用浏览器打开的本地文件管理器，专治「用不惯 Mac 访达」的各种不爽。

## 你是否也有这些困扰？

Mac 的访达（Finder）对很多从 Windows/Linux 转过来的用户来说，体验并不友好：

- **点击区域太小**：访达的文件项点击热区有限，经常点了没反应，需要精确点到文件名或图标上
- **缺少地址栏直接编辑**：无法像在 Windows 资源管理器里那样直接在路径栏输入路径跳转
- **操作效率低**：右键菜单层级深，常用操作（新建文件夹、重命名、删除）不够直观
- **文件预览受限**：代码文件、配置文件等文本文件预览能力有限
- **隐藏文件切换麻烦**：每次都要记 `Cmd + Shift + .` 快捷键
- **多选操作不便**：Ctrl+Click、Shift+Click 多选体验不如 Windows 流畅
- **没有剪切功能**：访达的移动文件需要拖拽或用 `Cmd+Option+V`，不够直觉
- **多窗口管理混乱**：开多了访达窗口，找文件变成窗口捉迷藏

如果你也有同感，这个项目就是为你准备的。

## 这是什么？

**Local File Manager** 是一个运行在本地 `localhost` 的文件管理器，用浏览器打开即可使用。

不安装任何客户端，不上传任何文件到云端——它只在你自己的电脑上运行，管理你自己的文件。

完全对标 Windows 资源管理器的操作习惯，让从 Windows 转过来的用户无缝上手。

## 功能一览

### 文件操作

| 功能 | 说明 | 快捷键 |
|------|------|--------|
| 新建文件夹 | 在当前目录创建新文件夹 | `Ctrl+Shift+N` |
| 新建文件 | 在当前目录创建空文件 | 右键菜单 |
| 复制 | 复制选中的文件/文件夹到剪贴板 | `Ctrl+C` |
| 剪切 | 剪切选中的文件/文件夹到剪贴板 | `Ctrl+X` |
| 粘贴 | 将剪贴板内容粘贴到当前目录，冲突自动重命名 | `Ctrl+V` |
| 重命名 | 内联编辑文件名，自动选中文件名（不含扩展名） | `F2` |
| 删除 | 删除选中的文件/文件夹，支持批量删除 | `Delete` |
| 上传文件 | 工具栏按钮或拖拽文件到页面上传 | 拖拽上传 |

### 选择操作

| 功能 | 说明 |
|------|------|
| 单击选择 | 点击文件选中，取消其他选中 |
| Ctrl+Click | 切换单个文件的选中状态（非连续多选） |
| Shift+Click | 从上次选中项到当前项的连续区间选择 |
| Ctrl+A | 全选当前目录所有文件 |
| 点击空白区域 | 取消所有选择 |
| 状态栏显示 | 选中数量 + 选中文件总大小 |

### 导航

| 功能 | 说明 | 快捷键 |
|------|------|--------|
| 面包屑导航 | 路径分段可点击跳转 | — |
| 地址栏编辑 | 双击面包屑切换为可编辑地址栏 | `Alt+D` |
| 后退 | 基于浏览历史后退 | `Alt+←` / `Backspace` |
| 前进 | 基于浏览历史前进 | `Alt+→` |
| 上级目录 | 导航到父目录 | `Alt+↑` |
| 刷新 | 刷新当前目录 | `F5` |

### 右键菜单

**文件/文件夹上右键：**
- Open (打开)
- Cut (剪切) / Copy (复制)
- Rename (重命名) / Delete (删除)
- Properties (属性)

**空白处右键：**
- New Folder (新建文件夹) / New File (新建文件)
- Paste (粘贴) — 剪贴板有内容时显示
- Select All (全选)
- Refresh (刷新)

### 视图与排序

| 功能 | 说明 |
|------|------|
| 列表视图 | 详细信息：名称、大小、类型、修改时间 |
| 网格视图 | 大图标网格布局 |
| 列排序 | 点击表头按名称/大小/类型/修改时间排序，支持升降序 |
| 隐藏文件 | 一键切换显示/隐藏文件 |
| 文件预览 | 文本/代码文件内容预览，图片直接展示 |
| 属性对话框 | 查看文件名、类型、位置、大小、创建/修改时间 |

### 键盘快捷键速查

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+A` | 全选 |
| `Ctrl+C` | 复制 |
| `Ctrl+X` | 剪切 |
| `Ctrl+V` | 粘贴 |
| `Ctrl+Shift+N` | 新建文件夹 |
| `Delete` | 删除选中项 |
| `F2` | 重命名 |
| `F5` | 刷新 |
| `Alt+D` | 编辑地址栏 |
| `Alt+←` / `Backspace` | 后退 |
| `Alt+→` | 前进 |

| `Escape` | 取消选择 / 关闭弹窗 |

## 快速开始

### 环境要求

- Python 3.8+
- macOS / Linux / Windows

### 安装与运行

```bash
# 1. 克隆仓库
git clone https://github.com/zjverse/local-file-manager.git
cd local-file-manager

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动
python3 main.py
```

浏览器打开 **http://127.0.0.1:4060** 即可使用。

默认从你的 Home 目录开始浏览。

## 技术栈

- **后端**: Python + FastAPI
- **前端**: 原生 HTML/CSS/JS（零构建工具，零框架依赖）
- **翻译层**: 独立的 `i18n.js`，通过 MutationObserver 运行时注入，不侵入原始代码

## 项目结构

```
local-file-manager/

├── requirements.txt   # 依赖

│   ├── style.css      # 样式

│   └── i18n.js        # 独立翻译层

```

## 安全

- 仅在 `127.0.0.1` 监听，不暴露到局域网
- 文件操作不经过任何云端，所有数据留在本地
- 删除操作有二次确认弹窗
- 禁止删除 Home 根目录
- 复制/移动冲突自动重命名，不覆盖已有文件
